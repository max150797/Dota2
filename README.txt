Проект предсказывания исходов матчей по игре Dota 2

В данном проекте содержится:
	1. Парсер, который позволяет извлечь с сайта https://dota2.ru/esport/matches информацию по сыгранным матчам. "Parser_dota2"
	2. Notebook, в котором содержится обработка исходного датасета и модель машинного обучения
